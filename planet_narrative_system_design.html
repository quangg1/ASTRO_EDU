② Core abstraction — NarrativeSpace (3 lớp đúng, nhưng tên + scope cần rõ hơn)

LAYER 1 — WorldConfig 
// Tách planet khỏi narrative
interface WorldConfig {
  bodySlug: 'earth'|'mars'|'venus'|string
  atmospherePreset: string
  // 'modern'|'archean'|'olympus-dust'|'venus-haze'
  colorGrade: string
  effectTags: string[]
  // 'volcano'|'meteor'|'dust-storm' → effect registry
  lightingPreset: string
  // không hard-code effect, chỉ tag → renderer lookup
}
LAYER 2 — SequenceConfig 
// "Spine" là sequence — không bắt buộc là time
interface SequenceConfig {
  type:
    | 'geologic_ma'     // Earth, Mars geology
    | 'mission_sols'    // Mars rover timeline
    | 'chapters'        // pure narrative, no time
    | 'custom_epochs'   // teacher-defined units
  unit: string          // "Ma"|"Sol"|"Chapter"|"Epoch"
  range?: [number, number]
  direction: 'forward'|'reverse'
  // Earth: 4500 → 0 (reverse). Mission: 0 → n (forward)
}
LAYER 3 — Beat (content atom) 
// EarthStage → Beat (generalized)
interface Beat {
  id: string
  ref: string|number   // Ma / Sol / Chapter#
  label: Record<Lang,string>
  description: Record<Lang,string>
  worldOverride?: Partial<WorldConfig>
  // override atmosphere/effects tại beat này
  assets: BeatAsset[]    // organisms, markers, models
  interactions: Interaction[]
  lessonLinks: LessonRef[]
}
// NarrativeSpace — the top-level document
interface NarrativeSpace {
  id, slug, version:    string
  title:                 Record<Lang,string>
  templateId:           string       // preset nào đã dùng
  world:                 WorldConfig
  sequence:             SequenceConfig
  beats:                 Beat[]
  assetBundle:           AssetBundle  // replaces iconicOrganisms implicit contract
  meta: {
    verified:            boolean      // teacher custom vs verified preset
    sourceRefs:          Citation[]   // bắt buộc cho scientific accuracy
    author, createdAt, publishedAt
  }
}

EARTH HISTORY = PRESET, KHÔNG PHẢI ENGINE
earthHistoryData.ts hiện tại → convert thành 1 NarrativeSpace JSON với templateId='deep-time-journey'. Không xóa, chỉ wrap lại.
iconicOrganisms implicit contract → thành AssetBundle field trong NarrativeSpace. Beat gắn asset qua id, không phải stage index 0-21.
Teacher tạo "Mars Geology" = chọn template 'deep-time-journey' → đổi bodySlug='mars' → sequence type='custom_epochs' với Noachian/Hesperian/Amazonian → điền beats.
3 themes on Mars = template 'topic-chapters' với 3 beats, bodySlug='mars' — không cần timeline.
Earth History không mất đi tính unique — nó chỉ là NarrativeSpace đầu tiên với verified=true, sourceRefs đầy đủ nhất. Mọi thứ teacher tạo sau đều follow cùng schema.

⑤ 3 quyết định kiến trúc quan trọng nhất

✓ AssetBundle thay implicit contract
iconicOrganisms gắn với beat qua id, không index 0-21. Thêm/xóa beat không làm lệch organisms. Beat tự mang assetRefs[], không depend external array.
✓ verified vs teacher-custom
NarrativeSpace có meta.verified flag. Verified spaces (Earth History gốc) không thể bị sửa bởi teacher — chỉ fork. Teacher custom spaces có sourceRefs bắt buộc để maintain scientific accuracy.
✓ Effect là tag, không phải code
effectTags: ['volcano', 'dust-storm'] → renderer lookup effect registry. Thêm effect mới = thêm vào registry, không sửa NarrativeSpace schema hay component code. Teacher chọn effect từ dropdown, không biết code.
